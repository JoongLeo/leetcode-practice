# leetcode-practice
记录 LeetCode 练习与题型归类，按主题文件夹组织，便于持续扩增与检索。

## 目录结构
约定：根目录按主题建文件夹，主题内按小类或技巧拆分；题目文件使用「编号 + 题名」命名。

示例结构（当前与建议保持一致）：
```
leetcode-practice/
  README.md
  滑动窗口与双指针/
    1-定长滑动窗口/
      1456. 定长子串中元音的最大数目.cpp
    2-不定长滑动窗口/
```

## 命名规范
- 主题目录：`主题名` 或 `NN-主题名`（建议统一为带序号，便于排序）
- 子目录：`N-小类/技巧`（如 `1-定长滑动窗口`）
- 题目文件：`题号. 题名.扩展名`（如 `1456. 定长子串中元音的最大数目.cpp`）

## 新增内容流程
1. 选择或新增主题目录（必要时添加序号）。
2. 在主题下新建小类目录（如 `1-定长滑动窗口`）。
3. 新建题目文件并按命名规范保存。
4. 可在主题目录下补充 `README.md` 作为题单索引（可选）。
